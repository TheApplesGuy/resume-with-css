Description

The project, lex.c, is from my system software class and it focused on lexical analysis using the PL/0 programming language. This project words by looking at certain keywords and symbols, and tokenizing them so that the program can be better organized into a structure. This allows for a compiler to have an easy time reading and processing the program.
